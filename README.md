# 🏠 Home Automation & Security System

This project demonstrates a **Home Automation and Security System** designed to automate household devices and enhance home security.

The system allows users to monitor and control devices such as lights, sensors, and alarms to improve convenience and safety.

---

## 🎯 Project Objectives

* Automate home appliances
* Improve home security
* Monitor home environment using sensors
* Provide an efficient and scalable automation solution 

---

## ⚙️ Features

* Control home appliances
* Motion detection
* Security alert system
* Real-time monitoring
* Energy-efficient automation

---

## 🧰 Components Used

* Microcontroller (Arduino / ESP / similar)
* Ultrasonic Sensor
* Motion Sensor
* Relay Module
* Buzzer
* LEDs
* Connecting wires
* Power supply
